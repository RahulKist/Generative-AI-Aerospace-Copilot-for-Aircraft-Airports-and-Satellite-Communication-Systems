# Generative-AI-Aerospace-Copilot-for-Aircraft-Airports-and-Satellite-Communication-Systems


This project demonstrates an AI-powered aerospace engineering assistant for aircraft engine health monitoring.

## Features

- NASA C-MAPSS Turbofan Dataset
- Remaining Useful Life (RUL) Prediction using LSTM
- Digital Twin Integration
- Retrieval-Augmented Generation (RAG)
- Engineering Document Retrieval
- AI-Based Maintenance Recommendations
- Explainable Decision Support

## Technologies

- Python
- TensorFlow / Keras
- Scikit-learn
- LangChain
- ChromaDB
- Sentence Transformers
- Google Gemini API
- Gradio

## Workflow

Sensor Data → LSTM → RUL Prediction → Digital Twin → RAG → AI Maintenance Recommendation
